Phân tích bước đầu tiên

Xác suất chạm:
\(g_l(k) = \sum_m P_{k,m}\, g_l(m)\)
Biên: \(g_l(k) = \ind_{\{k=l\}}\) khi \(k \in A\)

Thời gian chạm TB:
\(h_A(k) = 1 + \sum_l P_{k,l}\, h_A(l)\)
Biên: \(h_A(l) = 0\) khi \(l \in A\)

Dạng ma trận:
\(\mathbf{g}_l = (I-Q)^{-1} R\, \mathbf{e}_l\)
\(\mathbf{h}_A = (I-Q)^{-1} \mathbf{1}\)

Bài toán con bạc

Khi \(p \neq q\), đặt \(\rho = q/p\):
\(g(i) = \dfrac{1-\rho^i}{1-\rho^N}\)

Khi \(p = q = 1/2\):
\(g(i) = i/N\)
