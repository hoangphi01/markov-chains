grid_on Ma trận chuyển P

\(P_{i,j} = \PP(Z_{n+1} = j \mid Z_n = i)\)

Hàng \(i\) = trạng thái hiện tại, cột \(j\) = trạng thái kế tiếp. Đọc theo hàng để biết "từ \(i\) đi đâu".

Tổng hàng = 1

\(\sum_{j} P_{i,j} = 1\)

Xác suất phải phân bổ hết — từ trạng thái \(i\), chắc chắn phải đi đâu đó.

Lũy thừa ma trận

\(P^k_{i,j} = \PP(Z_{n+k} = j \mid Z_n = i)\)

Nhân \(P\) với chính nó \(k\) lần để biết xác suất chuyển sau \(k\) bước.

Chapman–Kolmogorov

\(P^{m+n} = P^m \cdot P^n\)

Chia hành trình thành hai phần: đi \(m\) bước rồi đi thêm \(n\) bước.

swap_horiz Chuỗi 2 trạng thái

\(P = \begin{bmatrix} 1-a & a \\ b & 1-b \end{bmatrix}\)

\(a\) = xác suất rời trạng thái 0, \(b\) = xác suất rời trạng thái 1.

Trị riêng

\(\lambda_1 = 1\), \(\lambda_2 = 1 - a - b\)

Phân phối giới hạn

\(\pi_0 = \dfrac{b}{a+b}\), \(\pi_1 = \dfrac{a}{a+b}\)

Trạng thái nào khó rời (\(a\) hoặc \(b\) nhỏ) thì chiếm tỷ lệ cao hơn khi ổn định.

Tốc độ hội tụ

\(\sim |\lambda_2|^n = |1 - a - b|^n\)

\(|1 - a - b|\) nhỏ → hội tụ nhanh. Khi \(a + b = 1\): hội tụ ngay lập tức.
