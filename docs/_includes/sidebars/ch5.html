So sánh P và Q

\(P\)\(Q\)
Ngoài chéo\(\geq 0\)\(\geq 0\)
Đường chéo\(\geq 0\)\(\leq 0\)
Tổng hàng\(= 1\)\(= 0\)

\(P\) = "đi đâu", \(Q\) = "đi nhanh thế nào"

Phân phối mũ

\(\PP(X > t) = e^{-\lambda t}\), \(\E[X] = 1/\lambda\)

Không nhớ:
\(\PP(X > s+t \mid X > s) = \PP(X > t)\)

Min: \(\min(X_1,...,X_n) \sim \text{Exp}(\sum \lambda_i)\)

Cạnh tranh:
\(\PP(X_1 < X_2) = \frac{\lambda_1}{\lambda_1+\lambda_2}\)

Công thức chính

\(P(t) = e^{tQ}\)
Kolmogorov: \(P'(t) = P(t)Q = QP(t)\)

Phân phối dừng: \(\pi Q = 0\)

Chuỗi nhúng:
\(\hat{P}_{i,j} = \dfrac{\lambda_{i,j}}{-\lambda_{i,i}}\)
